# U.S. airport coverage map attribution

The coverage graphic is a generated, presentation-only asset for Flight Delay Risk.

- U.S. boundary and state geometry: U.S. Census Bureau data distributed through `us-atlas` 3.0.1 (ISC).
- Airport locations: OpenFlights / OurAirports-derived records distributed through `@nwpr/airport-codes` 3.0.3 (ISC package; source database attribution retained here).
- The plotted set is restricted to the 348 airport codes present in the committed Flight Delay Risk schedule context.

No third-party runtime dependency is required. The generated SVG is committed as a static asset.
